CSR-8

1 Scope

The management of electronic transport regulations (METR) provides trustworthy, authoritative, machine-interpretable, transport-related rules for using surface transport facilities.

The scope of METR includes both rules that are relatively static (e.g., static speed limits) as well as those that are dynamic (e.g., variable speed limits, lane closures at incident scenes). However, where appropriate, METR is expected to incorporate existing standards (e.g., ISO/TS 19091 for signalized intersections).

This document defines the requirements for a METR component system that is responsible for maintaining copies of relevant transport-related rules for a METR user and reporting any detected discrepancies among rules. This document is presented in a format that is consistent with the guidance provided by ISO 29148 [2] for a system requirements document. 

For an understanding of how the METR user system fits into the larger METR system of systems, see ISO 24315-3. The operational concept for METR is provided in ISO 24315-2.
